# 安装

## windows

 1. 搜索docker desktop 安装

    按照提示安装，可能需要安装linux升级核心

 2. 控制面板->程序->启动或关闭Windows功能  打开hyper-v，重启

## macos

	1. 搜索docker desktop 安装
